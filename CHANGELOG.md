# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-05

### Added
- **Full Habit Tracking**: Support for creating daily, weekly, and monthly habits with custom frequencies.
- **Reports Dashboard**: Analytics for habit completion, streaks, and "perfect days".
- **Authentication**: Secure user registration and login using JWT and bcrypt.
- **Bilingual Support**: Full support for Indonesian (ID) and English (EN) languages.
- **Responsive Design**: Mobile-friendly user interface with a collapsible sidebar.
- **Feedback System**: Integrated form for user feedback submission.
- **Backend API**: Robust Express.js API served with PostgreSQL and Prisma.

### Initial Release
- Initial release of BIKE (Bicker) - Habit Tracker Application.
